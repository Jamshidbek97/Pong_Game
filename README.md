# Pong_Game

#Controls

Arrow Keys for Direction:
#For Player A:
To move paddle up press 'w'
To move paddle down press 's'

#For Player B:
To move paddle up press 'UP_ARROW'
To move paddle down press 'DOWN_ARROW'

#Dependencies:
Python v3.x is required.
turtle module is also required.
